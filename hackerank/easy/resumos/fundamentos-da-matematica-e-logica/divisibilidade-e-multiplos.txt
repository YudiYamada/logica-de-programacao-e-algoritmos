Resumo - Divisibilidade e Múltiplos em JavaScript

1) Divisibilidade
- Um número A é divisível por B se o resto da divisão (A % B) for igual a 0.
- Exemplo:
  10 % 2 === 0   // true → 10 é divisível por 2
  15 % 4 === 0   // false → 15 não é divisível por 4
- Função genérica:
  function ehDivisivel(a, b) {
    return a % b === 0;
  }

2) Múltiplos
- Um número A é múltiplo de B se existe um inteiro k tal que A = B * k.
- Exemplo:
  12 é múltiplo de 3 → 12 = 3 * 4
  20 é múltiplo de 5 → 20 = 5 * 4
- Verificação em JS:
  function ehMultiplo(a, b) {
    return a % b === 0;
  }

3) Encontrar múltiplos em listas
- Exemplo: todos os múltiplos de 3 em um array.
  const numeros = [2, 3, 4, 6, 9, 12];
  const multiplosDe3 = numeros.filter(n => n % 3 === 0); // [3, 6, 9, 12]

4) Casos especiais
- Divisão por zero não é permitida:
  a % 0 → NaN
- Zero é múltiplo de qualquer número:
  0 % n === 0 (para n ≠ 0)

5) Aplicações práticas
- Verificar se um número é par ou ímpar:
  n % 2 === 0 → par
  n % 2 !== 0 → ímpar
- Encontrar múltiplos em sequências (ex: múltiplos de 5 até 50).
  for (let i = 5; i <= 50; i += 5) {
    console.log(i);
  }

Resumo:
- Use o operador módulo (%) para verificar divisibilidade.
- Múltiplos são obtidos multiplicando um número por inteiros sucessivos.
- Zero é múltiplo de qualquer número, mas não pode ser divisor.
